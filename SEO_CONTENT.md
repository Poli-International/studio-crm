#### H2: Frequently Asked Questions (FAQ)

**Q1: Where is my client data stored?**
All data is stored in your browser's localStorage. It never leaves your computer or device. No external servers are involved.

**Q2: Can multiple artists use this CRM simultaneously?**
Currently, this is a local-only tool. Each artist or studio must manage their own local database on their device.

**Q3: How do I backup my client records?**
Use the Export Data feature to save a JSON file to your secure external drive. Regular backups are recommended.

**Q4: Is this CRM GDPR compliant?**
Yes. You have full data sovereignty with consent tracking, right-to-erasure support, and data portability features built in.

**Q5: Can I track ink and needle inventory?**
Yes. The Inventory module allows you to track supplies with quantities, set minimum thresholds, and receive low-stock alerts.

**Q6: Does the tool include aftercare email templates?**
Yes. There are pre-built email templates for tattoo welcome, 3-day check-in, 7-day aftercare, 6-month touch-up, fresh piercing, and 2-week piercing check-up.

**Q7: Can I upload photos of client work?**
Yes. The Gallery module supports photo uploads with artist attribution, healing progress tracking, and tag categorization.

**Q8: What happens if I clear my browser data?**
Clearing browser data will reset the database. Always use the export feature to backup records before clearing browser data.

**Q9: Can I embed this tool on my studio website?**
Yes. The tool provides an embed code that allows you to integrate it into any website via iframe.

**Q10: Is there a mobile app version?**
The tool is a web application that works in mobile browsers, but there is no dedicated native mobile app at this time.
